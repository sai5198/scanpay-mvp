# ScanPay Payment Gateway

A modern payment gateway system that enables customers to scan products using barcodes and complete payments through QR codes, with integrated merchant verification.

## Features

- **Customer Interface**: Scan products, manage cart, generate payment QR codes
- **Merchant Dashboard**: Verify customer payments and manage orders
- **Admin Panel**: System administration and monitoring
- **Real-time Updates**: Live cart totals and order status tracking

## Quick Start

1. Open `index.html` in your browser for the customer interface
2. Open `merchant.html` for the merchant dashboard
3. Open `admin.html` for the admin panel
